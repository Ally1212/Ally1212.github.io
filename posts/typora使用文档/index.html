第零步 注意事项

中文标点符号不被Markdown所认可, 而且无法被转义. 所以应该使用英文标点. 并且在使用后加一个空格使其更美观.

第一步 基础语法

1.加粗

文字两边都加上**

或者选中文字按下Ctrl +B

2.斜体

文字两边都加一个*

或者选中文字按下Ctrl+I

3.删除线

文字两边都加上~~

4.高亮

文字两边都加上==

5.代码

按三个```并按回车

` 在键盘左上角, 记得英文状态下

#include <stdio.h>
int main(){
	printf("hello world!");
	return 0;
}

6.引用

“>"+空格

21岁, 是学生

7.无序列表

“-"+ 空格

  • 一颗是枣树
  • 另一颗也是枣树

8.有序列表

数字 + “.” + 空格

  1. 垃圾
  2. 还是垃圾

9.上标

用”^“包围

10.下标

用”~“包围

H~2~O

11.链接

语法：[点击跳转正大光猫的博客网站]+(地址)

markdown本身不支持在里面打开链接，写在网站可以

百度

12.任务列表

“-” + [ ] 来使用

  • 刷B站

13.表格

语法：|分割 查看源码 删掉语句之间的空隙

或者右键直接插入表格

名字 性别 生日

14.图片

!+[图片名称]+(图片地址)——引用图片

怎么引用网络图片，点击浏览器审查元素，找到对应图片地址，引用即可
